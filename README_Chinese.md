# 日频量化框架

这是一个利用量价数据以及基本面数据进行截面收益率预测的项目。该项目额外探索小额量化投资的可行性。QBG是Quant HBG的意思。

## 整体框架

该框架主要分为几个部分，自动因子挖掘，因子聚合模型，回测框架。

### 自动因子挖掘

我们将实现一个基于因子表达式的自动因子搜寻框架，主要使用公式树的变异来构造因子。

### 因子聚合模型

主要包括分层模型，LASSO，树方法的boosting。
